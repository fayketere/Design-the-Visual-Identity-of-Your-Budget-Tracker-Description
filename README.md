# Budget Tracker

## Overview

This project is a simple Budget Tracker that allows users to record expenses and view them in a structured table.

This week's assignment focused entirely on improving the visual appearance of the existing project using CSS.

---

## Features

### 1. Color Palette

A professional blue and white color palette was applied throughout the application.

Used for:

- Page background
- Headings
- Buttons
- Table headers
- Cards

---

### 2. Typography

Google Fonts were added:

- Merriweather for headings
- Poppins for body text

These fonts improve readability and create a professional visual hierarchy.

---

### 3. Form Styling

The Add Expense form includes:

- Rounded input fields
- Padding
- Borders
- Focus effects
- Styled submit button
- Consistent spacing

---

### 4. Expense Table Styling

The table includes:

- Styled table header
- Borders
- Cell padding
- Alternating row colors
- Hover effect

---

### 5. CSS Box Model

The project uses:

- Margin to separate cards
- Padding inside each section
- Borders for structure
- Border radius for modern appearance
- Box shadows to create visual cards

---

## Technologies Used

- HTML5
- CSS3
- Google Fonts
